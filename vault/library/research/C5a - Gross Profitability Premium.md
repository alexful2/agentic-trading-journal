# C5a — Gross Profitability Premium

> Empirical-priors research note (research/ catalog). Not investment advice.

**Claim.** A simple, "clean" profitability ratio — gross profits to assets — predicts the cross-section of stock returns with roughly the same power as book-to-market value.

**Sources.**
- Novy-Marx (2013), "The Other Side of Value: The Gross Profitability Premium," *Journal of Financial Economics* 108(1):1–28. DOI 10.1016/j.jfineco.2013.01.003 *(citation of record)*. NBER WP 15940 (different title — see Tension).

**Evidence setting.** Gross profitability = (revenues − COGS) / assets. US equities, **July 1963–Dec 2010**, financial firms excluded. Long-short sorts and Fama-MacBeth cross-sectional regressions.

**Replication & decay.** Raw high-minus-low (most- minus least-profitable) spread **0.31%/month (t = 2.49)**; the Fama-French 3-factor *alpha* is larger at **0.52%/mo (t = 4.49)** because the strategy carries a negative value loading; the value-controlled profitability factor (PMU|BM) earns **0.48%/mo (t = 5.35)**. In Fama-MacBeth regressions the gross-profitability slope (t = 5.49) is comparably significant to log(B/M) (t = 5.98). Profitability and value are *negatively* correlated, so the two together hedge and strengthen each other.

**What this is NOT.** A cross-sectional portfolio result — **NOT a single-name buy/sell signal.** Surviving the [[C2 - Replication Crisis in Factor Anomalies]] gate (profitability is one of the categories that replicates) means it's a usable *prior*, not a trigger.

**How to apply.** Use it as a burden-of-proof screen, especially for capital-intensive, often-unprofitable names. The prior: profitable firms have outperformed unprofitable ones for the same valuation, so an unprofitable name needs a *specific* reason it's the exception (e.g., contracted offtake, a clear path to gross-profit positivity) — not just a thesis.

**Decision-rule translation.**
- For an unprofitable holding, write the explicit answer to "why is this name the exception to the profitability prior?" If you can't, the position carries extra burden.
- Use gross profitability (not net margins / EPS, which are noisier and more manipulable) when comparing names — that's the variable with the cleanest empirical signal.
- Pair with [[C13 - Asset Growth and Capex Caution]]: profitable *and* not over-investing is the durable combination; unprofitable *and* aggressively building is the riskiest.

**Tension / failure mode.** Early-stage infra build-outs can be deliberately unprofitable on GAAP gross margins while building genuine future earnings power — the factor would wrongly flag them. The factor is an *average* tendency; it does not adjudicate a specific thesis where the unprofitability is a known, financed, time-limited phase. Don't let it veto a name; let it raise the bar.
